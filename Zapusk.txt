## Основные команды для управления:

### Просмотр статуса:

```bash
docker-compose ps
```

### Остановка всех сервисов:

```bash
docker-compose down
```

### Запуск всех сервисов:

```bash
docker-compose up -d
```

### Перезапуск конкретного сервиса:

```bash
docker-compose restart mongodb  # только база данных
docker-compose restart backend  # только бэкенд
docker-compose restart frontend # только фронтенд
```

### Просмотр логов:

```bash
docker-compose logs mongodb  # логи базы данных
docker-compose logs backend  # логи бэкенда
docker-compose logs frontend # логи фронтенда
```
